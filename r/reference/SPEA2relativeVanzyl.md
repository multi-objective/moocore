# Results of SPEA2 with relative time-controlled triggers on Vanzyl's water network.

Results of SPEA2 with relative time-controlled triggers on Vanzyl's
water network.

## Usage

``` r
SPEA2relativeVanzyl
```

## Format

An object of class `data.frame` with 107 rows and 3 columns.

## Source

Manuel López-Ibáñez (2009). *Operational Optimisation of Water
Distribution Networks*. Ph.D. thesis, School of Engineering and the
Built Environment, Edinburgh Napier University, UK.
<https://lopez-ibanez.eu/publications#LopezIbanezPhD>.

data(SPEA2relativeVanzyl)
